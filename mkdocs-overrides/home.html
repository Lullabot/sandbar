Disposable Claude Code development VMs on Lima — spin one up in seconds, point Claude Code at a repository, and throw it away when you're done.

Get Started View on GitHub

  • Install and Go

    Install with Homebrew and you're done — the whole provisioning stack ships inside the binary, with nothing else to set up.

  • Fast, Disposable VMs

    A shared base image is built once, then each VM is a near-instant clone you can tear down without a trace.

  • A Board for Every VM

    Start, stop, reset, shell into, and watch your VMs from a terminal UI — or drive it all headlessly from the CLI.

  • Scoped Secrets

    Inject tokens and environment into a VM, globally or per-repository, without baking them into the base image.
